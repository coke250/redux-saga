# Redux-Saga 연습 레포지토리
- 랜덤으로 강아지 사진을 불러오는 과정을 통해 Redux-Saga를 학습합니다.
- `fetch()` 함수를 사용하여 API를 호출합니다.
- 강아지 사진을 불러오는 API는 [Dog CEO API](https://dog.ceo/dog-api/)를 사용했습니다. (~~시바견 사진은 있는데 진돗개 사진은 없음~~)
